摘要: HTML的DOCTYPE和div布局的基本用法

HTML的DOCTYPE

DOCTYPE一般来说，有三种用法

HTML4.01 Strict

该 DTD 包含所有 HTML 元素和属性，但不包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。

HTML 4.01 Transitional

该 DTD 包含所有 HTML 元素和属性，包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。

HTML 4.01 Frameset

该 DTD 等同于 HTML 4.01 Transitional，但允许框架集内容。

div布局

在我们编译一个网站时，首先要注意的是布局，通常我们用div来布局占块。

建议新手占块时给每一个div都设置background，这样你能清楚分辨出你写的每一个div。div的比较霸道，如果不设置width这样div的宽度会独占整个页面的宽度，但不能不设置高度。

那么如何让两个不太宽的div在同一行显示呢？

这时可以用到float（浮动）的属性，当div的宽度累积操过页面的宽度时，div会自动换行。

设置float属性时没给前面的div设置的话，这样不会出现float现象，如果没有给后面的div设置float，这样两个div会重叠，页面上会优先显示设置了float的div。

当一个或者多个div的高度累积超过浏览器页面的高度，就会出现所谓的滚动条（竖），横向滚动条不会由多个div宽度累积相加成，最多只会累积加到浏览器的默认宽度就会换行，横向滚动条只会由某个div的设置的明确的宽度大于浏览器时才会出现。
